How to use the program

  • Select the desired attbutes on the panel.
  • Adjust the weights of the selected attributes.
  • Click on a city on the map to inspect it in the city card on the right.
  • Select multiple cities on the map to compare them.
  • Use the radar chart on top to compare the city's fingerprints against each other.
  • Use the city card to view detailed information about each city.
  • Use the bar charts on the bottom to compare the selected attributes between the cities.
